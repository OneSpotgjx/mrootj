一加Fastboot工具箱 v2.0
 
一款专为一加（及OPPO/真我）设备打造的Termux端Fastboot工具集，集成依赖安装、设备管理、分区备份、镜像刷入等全套功能，支持A/B分区智能识别与操作，新手友好且高效稳定。
 
🌟 核心功能
 
- 📦 一键安装依赖：自动补全fastboot、adb、termux-usb等必备工具
- 🔍 智能设备检测：支持fastboot/ADB模式识别，自动读取A/B分区slot状态
- 🔓 灵活解锁方案：兼容新老机型Bootloader解锁命令，支持自定义指令
- 💾 分区管理：精准选择boot/init_boot分区，支持单slot/双slot备份
- 🚀 镜像刷入：支持指定slot刷入、双分区同步刷入、slot切换刷入等多种模式
- 📜 完整日志：操作全程记录，支持按日期、关键词查询
- ⚡ 一键自动流程：从依赖安装到镜像刷入全流程自动化，新手无忧
 
📋 支持设备
 
- 一加全系列设备（优先适配Android 12+机型）
- OPPO/真我系列设备（VID适配）
- 支持A/B分区与传统分区设备
 
🛠️ 前置要求
 
1.手机安装Termux应用（建议从官方渠道下载）
2.开启开发者选项与USB调试
3.授予Termux USB访问权限
4.设备电量≥50%（避免操作中断）
 
🚀 快速开始
 
1. 安装步骤
 
bash
  
# 1. 克隆仓库
git clone https://github.com/OneSpotgjx/mrootj.git
cd mrootj

# 2. 添加执行权限
chmod +x mrootjbl.sh

# 3. 运行工具
./mrootjbl.sh

 
2. 首次使用引导
 
1.运行后优先选择「1. 安装必要依赖」完成环境配置
2.通过「3. 帮助进入fastboot模式」将设备切入fastboot
3.依次完成「4. 选择设备」→「5. 选择VID」→「6. 解锁Bootloader」
4.选择对应分区类型（新机型选init_boot，老机型选boot）
5.备份原厂分区后，即可通过「9. 刷入镜像文件」完成刷机
 
3. 一键自动流程
 
选择主菜单「10. 一键自动流程」，将按以下顺序自动执行：
安装依赖 → 检测设备 → 选择设备 → 选择VID → 解锁Bootloader → 选择分区 → 备份分区 → 刷入镜像
 
📝 注意事项
 
- 解锁Bootloader会清除设备所有数据，请提前备份重要文件
- 刷入镜像前务必确认文件路径正确，避免刷错分区
- 操作过程中保持USB连接稳定，请勿中途断开
- 支持自定义VID，适配其他品牌设备（格式需为0xXXXX）
 
📂 目录说明
 
- fastboot_backup/：默认分区备份目录，文件命名包含设备ID、分区类型与时间戳
- fastboot_tool.log：操作日志文件，记录所有执行步骤与结果
 
🔧 常见问题
 
1.依赖安装失败：检查网络连接，尝试手动执行pkg update && pkg upgrade
2.设备无法识别：确认USB调试已开启，重新插拔USB线并授予权限
3.刷入失败：检查镜像文件完整性，确认分区类型与设备匹配
4.A/B分区检测失败：部分老机型不支持，选择传统刷入方式即可
